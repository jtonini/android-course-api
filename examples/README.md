
## Client Examples

### PHP Example

See `examples/php_client.php` for a complete PHP integration example showing:
- Health checks
- File upload/download
- Directory listing
- Quota management
- Batch operations

Usage:
```php
// Edit php_client.php and set your token
define('STUDENT_TOKEN', 'your_actual_token');

// Run
php examples/php_client.php
```
